# Project overview
This is a beginner-level Python program that asks the user for their name,
age, hobby and city name then displays a friendly personalized welcome message.
# Setup Instructions
1. Install Python 3 or above
2. Download or clone this repository
3. Open terminal or command prompt
4. Run the program using:
   python Dibya.py
## What I Learned
- How to take user input using input()
- How to store values in variables
- How to display output using print()
- How to use f-strings in Python
